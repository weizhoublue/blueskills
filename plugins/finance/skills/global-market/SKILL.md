# Skill: global-market

**采集指定周期内影响美股、纳斯达克、黄金、美债、能源等资产的信息，形成资产配置参考报告**

**如果没有明确的要求，把最终的报告内容直接输出到标准输出，并且在标准输出中禁止输出无关的内容和中间思考过程**

## 调用场景

**适用**
- 在用户明确要求调用本 skill 进行美股、黄金、美债等全球资产配置分析时，采集近期市场信息形成参考报告

**不适用**
- 用户未明确要求调用本 skill，禁止使用

## 执行流程

### 第 0 步：准备与初始化

1. **获取当前真实时间**：获取本地时区的当前真实时间（禁止使用模型截止时间，禁止凭记忆猜测)
2. **确认本地工具可用**：检查至少一个联网搜索工具。若不可用则终止流程。
3. **创建主题报告临时目录**：根据当前本地时间生成唯一标识 `<yymmddhhmmss>`，创建目录 `/tmp/global_market_reports_<yymmddhhmmss>/`，并将其记为 `report_dir`。

### 第 1 步：解析分析周期

根据用户输入生成本次分析周期。所有搜索请求必须带有明确的时间范围，不要只搜索“最新”。
- `event_start` = ``analysis_now` - 7 天
- `event_end` = `analysis_now`

常见周期映射：
- 最近一周 / 最近 7 天：analysis_now 往前 7 天
- 最近两周 / 最近 14 天：analysis_now 往前 14 天
- 最近一个月 / 最近 30 天：analysis_now 往前 30 天
- 本周：本周一 00:00 到 analysis_now
- 本月：本月 1 日 00:00 到 analysis_now
- 明确日期范围：使用用户指定开始和结束日期

### 第 2 步：各主题事实采集（主 Agent 顺序执行）

主 Agent 按以下主题顺序逐一采集信息，**禁止融入任何主观判断**。

**联网搜索工具使用原则**
- 所有联网搜索的行为，必须严格遵循如下调用优先级，当一个工具不可用时，再尝试使用低优先级的工具
  1. Tavily skill
  2. exa mcp
  3. firecrawl mcp
  4. 其他可用联网搜索工具
- 每个主题有明确的联网搜索次数上限

**主 Agent 执行指令**：
1. 所有报告内容均通过联网搜索获取, **禁止任何主观判断**
2. 每完成一个主题，立即将该主题完整 Markdown 报告写入 `report_dir/topic_<序号>.md`；文件只包含该主题报告内容。
3. 写入成功后，以临时文件为后续汇总的唯一来源，不依赖当前上下文保留主题内容。
